# 查询裸金属服务器网卡信息（OpenStack原生）<a name="ZH-CN_TOPIC_0053158678"></a>

## 功能介绍<a name="section36073588"></a>

查询裸金属服务器的网卡信息，比如网卡的MAC地址、私网IP信息。

## URI<a name="section56226836"></a>

GET /v2.1/\{project\_id\}/servers/\{server\_id\}/os-interface
